# Fetch Rewards Form Submission
- This app utilizes React, Bootstrap, and Axios 
- Submits a form to Fetch Rewards IF all inputs are filled out

## Installation 
- `cd` into the project directory 
- `npm install` to install the dependencies needed for the application
- `npm start` to start the server

```console
npm install
npm start
```
